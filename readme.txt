== Description ==
Плагин-виджет позволяет размещать на сайте скрытый текст. Поддерживает Text, HTML, CSS, JavaScript, Flash, PHP. Например Вы можете разместить ссылки из sape  и не испортить внешний вид сайта.
Плагин распространяется бесплатно!
Просьба оставлять ссылку на мой сайт, т.к. она не повредит.
